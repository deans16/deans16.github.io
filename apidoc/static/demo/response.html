1.普通-响应示例:

{
    "message": "",
    "code": 0,
    "data": 100,
    "extendInfo": [],
    "timeStamp": 0,
    "execTime": 0
}

data为有效可用数据,message只有当code>0时候有效,错误提示

code解释:

code 解释
0 通用正确返回
1 通用错误返回
1001 token失效
1002 token过期,可以通过认证服务的: [刷新token] 进行token置换
1003 访问频率超出
1004 服务器内部错误

2.列表数据-响应示例:

{
    "message": "",
    "code": 0,
    "data": {
        "list": [
            {
                "id": 68,
                "orderNo": "60ec07a0-4b1b-4f9f-b85a-47770f098fec",
                "agentId": 1,
                "userId": 1
            },....
        ],
        "count": 29
    },
    "extendInfo": [],
    "timeStamp": 0,
    "execTime": 0
}

data.list为返回【列表数据】,data.list为返回【列表条数】
